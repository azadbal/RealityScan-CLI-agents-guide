# RealityScan CLI Agents Guide

Single-file documentation for AI agents working with RealityScan / RealityCapture CLI workflows.

This repo contains:

* `RealityScan_CLI_combined_docs_for_agents.md` - a merged reference that combines the official CLI help content with an agent-oriented operating guide.

Use it as a compact handoff document for writing, reviewing, and debugging `.rscmd` / `.rccmd` automation, alignment, reconstruction, model export, and multi-instance orchestration tasks.
